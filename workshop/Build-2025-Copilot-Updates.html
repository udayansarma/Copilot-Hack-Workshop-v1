Generally Available

🤖 GitHub Copilot Coding Agent

  • Assign GitHub issues directly to Copilot — it works autonomously
  • Runs in secure, temporary dev environments via GitHub Actions
  • Pushes commits to draft pull requests for human review
  • Iterates based on PR feedback — responds to review comments
  • Full audit trail — every action is logged in session history
  • Respects branch protections, org policies, and security controls

🎭 Agent Mode in the IDE

Multi-file, multi-step coding from natural language

What It Does

  • Understands your entire codebase — not just the open file
  • Makes changes across multiple files in one session
  • Runs terminal commands, installs dependencies, fixes errors
  • Uses Plan Mode to architect before building
  • Supports Copilot Vision — image-to-code from design mockups

Three Modes

💬 Ask Mode

Read-only — explains code, answers questions

📋 Plan Mode

Designs architecture, creates file-by-file plan for approval

🤖 Agent Mode

Autonomous — makes changes, runs commands, iterates

GA

🔌 Model Context Protocol (MCP)

Give agents access to external tools, data, and APIs

How It Works

  • Configure MCP servers in your repo settings or VS Code
  • Agents can call external APIs, query databases, run browser tests
  • Built on an open standard — works across tools and platforms
  • Secure — respects org policies and requires explicit configuration

Available MCP Servers

🎭 Playwright

Browser testing — navigate pages, verify UI, take screenshots

🐙 GitHub

Create issues, manage PRs, search repos via API

🗄️ SQLite / Postgres

Query databases, analyze schemas, run migrations

New

🛠️ Agent Skills

Teach Copilot specialized, repeatable tasks

What Are Skills?

  • Folders with a SKILL.md instruction file + bundled assets
  • Include scripts, templates, reference docs, and examples
  • Agents automatically load relevant skills based on your prompt
  • Install from community: gh skills install github/awesome-copilot aspire

C#/.NET Skills Available

  • .NET Aspire — AppHost orchestration, service discovery
  • xUnit / NUnit / MSTest — testing patterns
  • Containerize ASP.NET — Docker multi-stage builds
  • NuGet Manager — dependency management

How Skills Differ from Instructions

📋 Instructions

Rules Copilot follows passively (naming, patterns, conventions). Applied to all interactions.

🛠️ Skills

Deep expertise Copilot uses actively (scripts, templates, reference data). Loaded on-demand when relevant.

Enterprise

🏢 Enterprise & Team Features

Governance & Control

  • Agent Rulesets — org-wide policies for what agents can/cannot do
  • One-toggle enablement — turn on Copilot features across entire org
  • Code Quality — AI-powered quality analysis beyond security
  • Audit logging — every agent action tracked and reportable
  • Branch protections honored — agents respect all existing policies

Security First

  • All agent PRs require human approval before merge
  • No CI/CD runs until code is reviewed
  • GHAS integration — CodeQL scans on agent-generated code
  • Secret scanning with extended metadata
  • Dependency review — automated security checks on new packages
  • Agent actions run in ephemeral environments — no persistent access
